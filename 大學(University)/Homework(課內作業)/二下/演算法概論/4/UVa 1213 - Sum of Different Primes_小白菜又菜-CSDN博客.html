UVa 1213 - Sum of Different Primes

小白菜又菜 2017-06-24 08:52:37 596 收藏 1
分類專欄： 解題報告 動態規劃（DP）

題目：計算整數n可以拆成k個不同的素數的拆分方法次數。

分析：動態規劃（DP）、數論。二維01背包。

            首先，打表計算1120內的素數；

            然後，利用二維01背包算法算法求解；

            狀態定義：f（k，i，j）為從前k個素數，取出j個不同的，組成數字i的方法數，prime[k]要取；

            轉移方程：f（k，i，j）= sum（f（k-1，i-prime[k]，j-1），f（k-2，i-prime[k]，j-1）， ...）；

            為了避免重取，將k放在最外層，去掉對應的下標節省狀態存儲空間。

說明：好久沒做Dynamic Programming了。

  1. #include <stdio.h>
  2. #include <stdlib.h>
  3.  
  4. int visit[1200] = {0};
  5. int prime[1200]; 
  6. int dp[1200][15] = {0};
  7.  
  8. int main()
  9. {
  10. 	int prime_count = 0;
  11. 	for (int i = 2; i <= 1120; ++ i) {
  12. 		if (!visit[i]) {
  13. 			prime[prime_count ++] = i;
  14. 			for (int j = i+i; j <= 1120; j += i) {
  15. 				visit[j] = 1;
  16. 			}
  17. 		}
  18. 	}
  19. 	
  20. 	dp[0][0] = 1;
  21. 	for (int k = 0; k < prime_count; ++ k) {
  22. 		for (int i = 1120; i >= prime[k]; -- i) {
  23. 			for (int j = 1; j <= 14; ++ j) {
  24. 				dp[i][j] += dp[i-prime[k]][j-1];
  25. 			}
  26. 		}
  27. 	}
  28. 	
  29. 	int n, k;
  30. 	while (~scanf("%d%d",&n,&k) && n) {
  31. 		printf("%d\n",dp[n][k]);
  32. 	}
  33. 	
  34.     return 0;
  35. }
